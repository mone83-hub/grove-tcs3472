Grove Farbsensor TCS3472 – MakeCode Erweiterung
Diese Erweiterung ermöglicht die Nutzung des Grove Color Sensor V2 (TCS3472) mit dem BBC micro:bit in Microsoft MakeCode.
Funktionen
⦁	Farbsensor initialisieren
⦁	Rotwert lesen
⦁	Grünwert lesen
⦁	Blauwert lesen
⦁	Helligkeit (Clear) lesen
Verwendung
1.	Erweiterung über Datei (ZIP) hinzufügen
2.	Sensor mit Grove I2C-Port verbinden
3.	Blöcke im Menü „GroveTCS3472“ verwenden
